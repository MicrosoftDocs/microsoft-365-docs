# Device inventory report

In the **Devices** view, you can select the **Export all** tab to download a comma-delimited file including this information:

- Serial Number
- Device Name
- Device Last check-in
- Managed By
- Manufacturer
- Model
- OS
- OS License
- OS Version
- Ownership
- Enrollment Date
- Ring Name
- Storage - Free in GB
- Storage - Total in GB
- Enrolled by User UPN
- Assigned User
- Status
- Age (Months)
- Profile
- Logged in Users
- TPM version
- Secure Boot Enabled
- Primary Disk Type
- Total Physical Memory
