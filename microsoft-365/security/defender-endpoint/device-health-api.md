## HTTP request

```http
GET /api/public/avdeviceshealth
```

## Request headers

| Name | Type | Description |
|:----|:----|:----|
| Authorization | String | Bearer {token}. **Required** |

## Request body

_Empty_

## Response

If successful, this method returns 200 OK with a list of device health details.
